# Employee Portal

A production-style Kubernetes demo application.

Tech Stack

- React Frontend
- Node.js Backend
- Docker
- Google Artifact Registry
- Google Kubernetes Engine (GKE)
- Helm
- Argo CD
- External Secrets Operator
- Google Secret Manager
- GKE Ingress

Deployment Flow

Developer
→ Bitbucket
→ Argo CD
→ Helm
→ GKE
→ Ingress
